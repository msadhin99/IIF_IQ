Core Row Rule

One row per paper, outcome, comparison/group, and time period. Split rows when outcomes, groups, follow-up periods, or comparison groups differ.

Controlled Values

Program DomainHealthcare; Education; Child Welfare; Criminal Justice. Add a relevant domain only when none fit.
Outcome DomainHealthcare; Education; Child Welfare; Criminal Justice; Employment.
Study TypeRCT; Quasi-Experimental; Simulation; Nonexperimental; WSIPP Meta Analysis; Observational/ Non-RCT; Group Study; Pre-post evaluation; Other.
SignificanceSignificant; Not significant; Not reported; Check notes.
Validation FlagNeeds check, or blank when no validator attention is needed.

Calculation Rules

  • Prefer raw fields: Control Baseline, Control Final, Treatment Baseline, Treatment Final.
  • Finals only Abs Diff = Treatment Final - Control Final.
  • 2x2 Abs Diff = (Treatment Final - Treatment Baseline) - (Control Final - Control Baseline).
  • Finals only Rel Change = (Treatment Final - Control Final) / Control Final.
  • Single-arm pre-post Rel Change = (Treatment Final - Treatment Baseline) / Treatment Baseline.
  • Full 2x2 Rel Change = (Treatment Final / Treatment Baseline) / (Control Final / Control Baseline) - 1.
  • If only modeled effects such as IRR, coefficient, odds ratio, cost difference, or effect size are reported and raw inputs are unavailable, record the reported effect in Notes and relevant error fields. Do not invent raw inputs.

Missing Data

  • Leave missing numeric cells blank. Never enter zero for missing values.
  • Use Not reported for Significance when no p-value, CI, or significance statement is reported.
  • Every numeric value needs a Source Location: page plus table, figure, appendix, exhibit, or section.
